Django installation

Install Python

To install Python on Windows you can download the latest Python release from here, you want the latest version of Python. Then once downloaded, run the executable and follow the given instructions. If you are using a linux distro with apt-get, you can install Python by navigating to terminal and entering the following commands:

sudo add-apt-repository ppa:jonathonf/python-3.6
sudo apt-get update
sudo apt-get install python3.6

If you are using Mac OS X, you need to follow a few steps. First, install a package manager- this tutorial will suggest Homebrew. Open terminal with GCC capabilities (here), and run

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Then, follow the script instructions. Now you need to insert Homebrew directory at the top of your PATH environment variable

export PATH="/usr/local/opt/python/libexec/bin:$PATH"
OR
export PATH=/usr/local/bin:/usr/local/sbin:$PATH

Now, install Python with

brew install python

Install Django

Thankfully, now that Python is installed, Django can be installed on all systems easily. To install django, run:

pip install Django==2.1.3
or
pip3 install Django==2.1.3

depending on the version of pip you have installed with your python distribution. Now Django should be installed and executable by Python! To create your first project, cd to where you want your project to be located, and type

django-admin startproject mysite

where mysite is any name you want to name your project. Its okay if you decide to change its location later, as long as you move all the project files Django shouldnâ€™t complain. Then type

python manage.py runserver

from the root directory of the newly created website, and navigate to the address it mentions, which should be localhost - http://127.0.0.1:8000/

If you see a landing page, then congratulations! You installed Django properly, and now you can begin your first Django project :)
